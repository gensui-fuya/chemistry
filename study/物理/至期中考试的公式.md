# 南昌高一（下）期中物理公式大全与注意事项

## 考试范围说明

江西省南昌市高中物理采用**人教版（2019）**教材。高一（下）期中考试主要考查**人教版必修第二册**中曲线运动、抛体运动、圆周运动、万有引力与航天以及功和能等相关内容。


## 一、运动学基础与曲线运动

### 1.1 运动学基本公式（匀变速直线运动）

| 物理量 | 公式 | 说明 |
|-------|------|------|
| 速度公式 | v = v₀ + a t | a 为加速度 |
| 位移公式 | x = v₀ t + ½ a t² | 位移矢量 |
| 速度-位移关系 | v² - v₀² = 2 a x | 不含时间 t |
| 平均速度 | v̄ = x / t = (v₀ + v) / 2 | 仅匀变速适用 |

### 1.2 竖直上抛/下抛运动

| 物理量 | 公式 |
|-------|------|
| 末速度 | vₜ = v₀ ± g t |
| 下落高度 | h = v₀ t ± ½ g t² |
| 上升时间 | t_上 = v₀ / g |
| 最大高度 | H = v₀² / (2 g) |
| 总时间（竖直上抛） | t_总 = 2 v₀ / g |

### 1.3 曲线运动

| 物理量 | 公式/结论 | 说明 |
|-------|------|------|
| 条件 | v₀ 与 F_合 不共线 | 速度与合力不在同一直线 |
| 速度方向 | 沿轨迹的切线方向 | 时刻变化 |
| 弯曲方向 | 从 v₀ 方向转向 F_合 方向 | 轨迹弯向合力一侧 |

**⚠️ 注意事项：**
- 曲线运动一定是**变速运动**，速度方向时刻在变化。
- 曲线运动中速度方向沿轨迹**切线**，轨迹弯向**合力方向**。
- 合力指向轨迹**凹侧**，力与速度的夹角决定速率变化（锐角→加速，钝角→减速）。


## 二、运动的合成与分解

| 类型 | 关键公式 | 说明 |
|------|---------|------|
| 最短时间过河 | t = d / v_船 ，船头正对河岸（α = 90°） | d 为河宽 |
| 最短路程过河 | cos θ = v_水 / v_船 ， s = d | 需 v_船 > v_水 |
| 绳拉船/杆问题 | 合运动 = 物体实际运动；分运动 = 沿绳伸缩 + 垂直绳摆动 | 速度分解原则：沿绳和垂直绳 |

**⚠️ 注意事项：**
- 合运动是物体的**实际运动**，分运动是等效分解。
- 船过河问题：**最短时间**与水流速度无关；**最短路程**需分析 v_船 与 v_水 的大小。
- 关联速度问题（绳、杆）：将实际速度沿绳和垂直绳方向正交分解，**沿绳方向速度大小相等**。


## 三、平抛运动

### 3.1 基本公式

以水平方向为 x 轴正向，竖直向下为 y 轴正向，抛出点为坐标原点。

| 物理量 | 公式 | 说明 |
|-------|------|------|
| 水平速度 | v_x = v₀ | 匀速直线运动 |
| 竖直速度 | v_y = g t | 自由落体运动 |
| 水平位移 | x = v₀ t | |
| 竖直位移 | y = ½ g t² | |
| 合速度 | v = √(v_x² + v_y²) = √(v₀² + (g t)²) | 矢量合成 |
| 速度偏角 θ | tan θ = v_y / v_x = g t / v₀ | 速度与水平方向夹角 |
| 位移偏角 φ | tan φ = y / x = g t / (2 v₀) | 位移与水平方向夹角 |
| tan θ 与 tan φ 关系 | tan θ = 2 tan φ | 重要推论 |
| 运动时间 | t = √(2 h / g) | h 为下落竖直高度 |
| 水平射程 | x = v₀ √(2 h / g) | |

### 3.2 推论

- **推论1**：任意时刻速度的反向延长线一定通过此时水平位移的**中点**。
- **推论2**：tan θ = 2 tan φ，即速度偏角的正切是位移偏角正切的**两倍**。
- **机械能守恒**：平抛运动中 E_k + E_p = ½ m v₀² + m g h 守恒（不计空气阻力）。

**⚠️ 注意事项：**
- 平抛运动是加速度 a = g 的**匀变速曲线运动**。
- 落地时间仅由**高度**决定：t = √(2h/g)，与初速度无关。
- tan θ = 2 tan φ 必须在**抛出点**建立坐标系时才成立。
- 与斜面结合时，要注意分析位移方向或速度方向与斜面的几何关系。
- 处理斜抛运动时，将其分解为水平匀速和竖直上抛运动。


## 四、圆周运动

### 4.1 描述圆周运动的物理量

| 物理量 | 定义式 | 单位 | 说明 |
|-------|--------|------|------|
| 线速度 | v = s / t | m/s | 方向沿切线 |
| 角速度 | ω = θ / t = 2π / T | rad/s | 描述转动快慢 |
| 周期 T | 转一圈所用时间 | s | |
| 频率 f | f = 1 / T | Hz | 单位时间转的圈数 |
| 转速 n | n = f = 1 / T | r/s | |

### 4.2 线速度与角速度的关系

v = ω r

v = 2π r / T = 2π r f = 2π r n

ω = 2π / T = 2π f = 2π n

### 4.3 向心加速度

a_n = v² / r = ω² r = 4π² r / T² = ω v

### 4.4 向心力

F_n = m v² / r = m ω² r = 4π² m r / T²

### 4.5 传动装置

| 传动类型 | 关系 |
|----------|------|
| 皮带传动 / 摩擦传动 | 边缘处**线速度大小相同**：v_A = v_B |
| 同轴转动 | **角速度相同**：ω_A = ω_B ， T_A = T_B |

### 4.6 离心运动与近心运动

- **离心运动**：F_供 < F_需 （合外力不足以提供所需向心力）或 F_供 = 0 ，物体做逐渐远离圆心的运动。
- **近心运动**：F_供 > F_需 ，物体做逐渐靠近圆心的运动。

**⚠️ 注意事项：**
- 匀速圆周运动是**变速曲线运动**：线速度方向、向心加速度方向不断变化。
- 向心力是**效果力**，由实际力（重力、弹力、摩擦力或合力）提供，受力分析时**不要画向心力**。
- 皮带传动：线速度相等；同轴转动：角速度相等。这是解题突破口。
- 竖直面内圆周运动（绳模型、杆模型）：
  - **绳模型**（最高点）：mg + F_N = m v² / r ，临界速度 v_min = √(g r) 。
  - **杆模型**（最高点）：临界速度 v_min = 0 ，杆可提供拉力或支持力。
- 汽车过拱桥：最高点压力 F_N = mg - m v² / r ；过凹桥：F_N = mg + m v² / r 。


## 五、万有引力与航天

### 5.1 开普勒三定律

| 定律 | 内容 |
|------|------|
| 第一定律（轨道定律） | 所有行星绕太阳运动的轨道都是**椭圆**，太阳在椭圆的一个**焦点**上 |
| 第二定律（面积定律） | 行星与太阳连线在相等时间内扫过**相等面积**（v₁ r₁ = v₂ r₂ 适用于近日点/远日点） |
| 第三定律（周期定律） | a³ / T² = k ， k 仅与**中心天体**有关 |

### 5.2 万有引力定律

F = G m₁ m₂ / r²

- G = 6.67 × 10⁻¹¹ N·m²/kg² ，引力常量，由**卡文迪许**扭秤实验测得。
- r 为两质点间的距离（或两球心间的距离）。

### 5.3 万有引力定律的应用（两条基本思路）

**思路1：万有引力提供向心力（环绕模型）**

G M m / r² = m v² / r = m ω² r = m (4π² / T²) r = m a

可推导出：

| 物理量 | 表达式 | 与 r 的关系 |
|-------|--------|-----------------|
| 线速度 | v = √(G M / r) | v ∝ 1/√r |
| 角速度 | ω = √(G M / r³) | ω ∝ 1/√r³ |
| 周期 | T = 2π √(r³ / (G M)) | T ∝ √r³ |
| 向心加速度 | a = G M / r² | a ∝ 1/r² |

**思路2：万有引力近似等于重力（表面模型）**

G M m / R² = m g

**黄金代换式**： G M = g R²

### 5.4 计算天体质量和密度

**方法一（利用表面重力加速度）**：
- M = g R² / G
- ρ = 3g / (4π G R)

**方法二（利用环绕天体的轨道半径和周期）**：
- M = 4π² r³ / (G T²)
- 近地卫星（r ≈ R）密度：ρ = 3π / (G T²)

**方法三**（利用天体半径和环绕速度）：
M = v² r / G ，再结合体积公式求密度。

### 5.5 宇宙速度

| 速度 | 数值 |
|------|------|
| 第一宇宙速度（环绕速度） | 7.9 km/s |
| 第二宇宙速度（脱离速度） | 11.2 km/s |
| 第三宇宙速度（逃逸速度） | 16.7 km/s |

第一宇宙速度推导：
v₁ = √(g R) = √(G M / R) ≈ 7.9 km/s

### 5.6 卫星变轨

| 过程 | 变化 |
|------|------|
| 从低轨到高轨 | 在低轨道某点**加速**→进入转移轨道（椭圆）→在高轨道某点**加速**→稳定运行 |
| 从高轨到低轨 | 在轨道某点**减速**→进入转移轨道→在目标轨道某点**减速**→稳定运行 |

**变轨规律**：低轨道上线速度更大，但机械能更小；同一位置加速则离开原轨道。

**⚠️ 注意事项：**
- 开普勒第三定律中 k 仅与**中心天体**质量有关，与环绕天体无关。椭圆轨道用**半长轴** a 代入。
- 万有引力定律公式中 r 是**两球心间距离**，地表物体 r = R_地 。
- **黄金代换式** G M = g R² 是连接天体表面重力加速度与轨道运动的关键。
- 同步卫星：周期 T = 24 h ，轨道在赤道平面内，高度约 36000 km。
- 卫星越远，v、ω、a 越小，T 越大（“高轨低速大周期”）。
- 变轨问题：同一位置速度突变导致轨道变化，但加速度不变（a = G M / r²）。


## 六、功和能

### 6.1 功

W = F s cos α

- α 为力 F 与位移 s 方向的夹角。
- α < 90°：正功；α = 90°：不做功；α > 90°：负功。
- 功是标量。
- 单位：**焦耳（J）**，1 J = 1 N·m。

### 6.2 功率

| 类型 | 公式 | 说明 |
|------|------|------|
| 平均功率 | P = W / t | 一段时间内做功快慢 |
| 瞬时功率 | P = F v cos α | 某一时刻做功快慢 |
| 机车恒定功率启动 | F = P / v ，v 增大→F 减小→a = (F - f)/m 减小→当 F = f 时达最大速度 | 最大速度 v_max = P / f |
| 机车恒定加速度启动 | F 一定→P = F v 随 v 增大→达额定功率后进入恒定功率阶段 | 最终最大速度同上 |

### 6.3 动能

E_k = ½ m v²

- 动能是标量，恒为非负。单位：**焦耳（J）**。
- 动能具有**瞬时性**，与速度大小平方成正比。

### 6.4 动能定理

W_合 = ΔE_k = E_k2 - E_k1 = ½ m v₂² - ½ m v₁²

- W_合 为所有外力做功的**代数和**（或合外力做的功）。
- 动能定理适用范围广：**直线/曲线、恒力/变力**均适用。

### 6.5 重力势能

E_p = m g h

- h 是相对于**零势能面**的高度。
- 势能是**标量**，有正负。重力势能是**物体和地球共有**的。
- 重力做功与路径无关，只与初末位置高度差有关。

**重力做功与重力势能变化的关系**：

W_G = E_p1 - E_p2 = - ΔE_p

### 6.6 弹性势能

E_p = ½ k x²

- x 为弹簧的**形变量**（压缩或伸长量）。
- 弹性势能与形变量的平方成正比。

### 6.7 机械能守恒定律

**内容**：在只有重力（或弹簧弹力）做功的系统内，动能与势能可以相互转化，总的机械能保持不变。

**条件**：W_其他 = 0 （除重力、弹力外其他力不做功或做功代数和为零）

**三种表达形式**：

| 表达形式 | 公式 | 说明 |
|----------|------|------|
| 守恒观点 | E_k1 + E_p1 = E_k2 + E_p2 | 需选取零势能面 |
| 转化观点 | ΔE_k = - ΔE_p | 动能增加量等于势能减少量 |
| 转移观点 | ΔE_A增 = ΔE_B减 | 适用于多物体系统 |

**⚠️ 注意事项：**
- **功的公式** W = F s cos α 中的 s 是**对地位移**，区分恒力做功与变力做功的处理方法。
- **功率公式** P = F v 中的 v 是瞬时速度对应瞬时功率；平均功率用总功除以总时间。
- 摩擦力做功：滑动摩擦力可以做正功、负功或不做功，但一对滑动摩擦力做功之和为负值（摩擦生热）。
- 计算重力势能必须先**确定零势面**，同一问题中零势面必须统一。势能值因零势面不同而不同，但势能变化量相同。
- **区分动能定理与机械能守恒**：
  - 动能定理：涉及**合外力总功**，任何情况均适用。
  - 机械能守恒：条件是其他力不做功；若题目涉及除重力外的其他力做功（如拉力、摩擦力），则不能直接使用机械能守恒。
- **判断机械能守恒的方法**：
  - 做功判断法：只有重力或弹力做功；
  - 能量转化判断法：只有动能与势能的相互转化，无机械能损失。
- 列机械能守恒方程时，务必**明确初、末状态**，并统一零势能参考面。
- 用动能定理时，W_合 必须是**所有力做功的代数和**（包括正功和负功），切勿漏力或多力。
