人教版高中物理 必修1 + 必修2（到万有引力）复习总整理
适用范围：必修1 + 必修2 前几章直到《万有引力与航天》
结构：概念 → 公式 → 易错点 → 注意事项

一、必修1：运动的描述与牛顿运动定律
1. 运动的描述
1.1 质点
概念：

用来代替实际物体的有质量的点

当物体的大小和形状对研究问题影响很小时，可视为质点

注意：

“小”不等于一定能看成质点，要看研究问题

例如研究地球公转时，地球可看成质点；研究地球自转时不能

1.2 参考系
概念：

描述运动时选作标准的物体

运动具有相对性

注意：

同一物体，选择不同参考系，运动状态可能不同

通常默认地面为参考系

1.3 位移与路程
位移：

从初位置指向末位置的有向线段

是矢量

路程：

物体实际运动轨迹的长度

是标量

注意：

位移只看初末位置，不看路径

路程一定不小于位移大小

1.4 速度
平均速度：

text
v = Δx / Δt
瞬时速度：

某一时刻或某一位置的速度

表示物体运动快慢和方向

速率：

瞬时速度的大小，是标量

注意：

平均速度不是平均速率

速度有方向，速率没有方向

1.5 加速度
text
a = Δv / Δt
含义：

表示速度变化快慢

不是“速度大”，而是“速度变化快”

注意：

加速度方向与速度变化量方向相同

速度大但加速度可以很小，甚至为 0

2. 匀变速直线运动
2.1 概念
匀变速直线运动：

加速度保持不变的直线运动

2.2 公式汇总
速度公式：

text
v = v0 + at
位移公式：

text
x = v0*t + 1/2*a*t^2
速度-位移公式：

text
v^2 - v0^2 = 2ax
平均速度公式（匀变速直线运动）：

text
v平均 = (v0 + v) / 2
位移与平均速度关系：

text
x = v平均 * t
2.3 自由落体运动
概念：

物体只在重力作用下从静止开始下落

特点：

初速度 v0 = 0

加速度为重力加速度 g

公式：

text
v = gt
h = 1/2*gt^2
v^2 = 2gh
h = v^2 / (2g)
注意：

近地面重力加速度常取 g ≈ 9.8 m/s^2

计算题中有时可取 g = 10 m/s^2，按题目要求来

3. 牛顿运动定律
3.1 牛顿第一定律
内容：

一切物体在没有受到外力作用时，总保持静止状态或匀速直线运动状态

核心理解：

力不是维持运动的原因，而是改变运动状态的原因

3.2 惯性
概念：

物体保持原来运动状态不变的性质

注意：

惯性只与质量有关

质量越大，惯性越大

3.3 牛顿第二定律
text
ΣF = ma
含义：

合力产生加速度

加速度与合力成正比，与质量成反比

注意：

公式中的 F 是合外力，不是某一个力

必须先选研究对象，再受力分析，再列式

3.4 牛顿第三定律
text
F_AB = -F_BA
含义：

作用力和反作用力大小相等、方向相反、作用在两个不同物体上

注意：

它们不能相互抵消，因为不作用在同一物体上

它们同时产生，同时消失

4. 常见力
4.1 重力
text
G = mg
方向竖直向下

作用点在重心

4.2 弹力
胡克定律：

text
F = kx
k：劲度系数

x：形变量

注意：

只适用于弹性限度内

方向总是阻碍物体形变

4.3 摩擦力
滑动摩擦力：

text
f = μN
μ：动摩擦因数

N：正压力

静摩擦力：

大小在一定范围内变化

最大静摩擦力通常满足：

text
fs_max = μs*N
注意：

摩擦力方向与相对运动趋势或相对运动方向相反

摩擦力不一定总是“阻碍运动”，也可能“帮助运动”

5. 受力分析与平衡
5.1 平衡状态
静止

匀速直线运动

5.2 平衡条件
text
ΣF = 0
注意：

在直线平衡中，各方向合力都为 0

常用正交分解法

5.3 受力分析注意点
先画物体受力图

只画物体受到的力

不漏力，不多力

明确摩擦力、支持力、拉力、重力方向

二、必修2：曲线运动、万有引力与航天
1. 曲线运动
概念：

运动轨迹是曲线的运动

特点：

速度方向沿轨迹切线方向

曲线运动一定是变速运动，因为速度方向在变

2. 平抛运动
2.1 概念
水平方向初速度为 v0

只受重力作用的运动

2.2 分解思想
水平方向：匀速直线运动

竖直方向：自由落体运动

2.3 公式
水平方向：

text
x = v0*t
vx = v0
竖直方向：

text
y = 1/2*g*t^2
vy = gt
合速度：

text
v = sqrt(vx^2 + vy^2)
速度方向：

text
tanθ = vy / vx = gt / v0
轨迹方程：

text
y = (g / (2v0^2)) * x^2
注意：

平抛运动中水平方向和竖直方向互不影响

下落时间只由高度决定，与水平速度无关

3. 圆周运动
3.1 线速度
text
v = 2πr / T
也可写成：

text
v = ωr
3.2 角速度
text
ω = 2π / T
3.3 向心加速度
text
ac = v^2 / r
ac = ω^2 * r
3.4 向心力
text
Fc = mv^2 / r
Fc = mω^2r
3.5 周期与频率
text
f = 1 / T
ω = 2πf
注意：

向心力不是一种独立的力，而是“指向圆心的合力”

向心力可以由重力、弹力、摩擦力、万有引力等提供

4. 万有引力定律
定律表达式：

text
F = G*m1*m2 / r^2
其中：

F：万有引力

G：万有引力常量

m1, m2：两个物体的质量

r：质心间距离

万有引力常量：

text
G = 6.67 × 10^-11 N·m^2/kg^2
适用条件：

对质点或球对称物体外部引力计算适用

天体之间的引力计算常可直接使用

注意：

距离 r 是质心之间的距离

不是物体表面间距离

5. 天体运动与开普勒定律
5.1 开普勒第一定律
行星绕太阳运动的轨道是椭圆，太阳位于一个焦点上

5.2 开普勒第二定律
行星与太阳连线在相等时间内扫过相等面积

5.3 开普勒第三定律
text
T^2 / r^3 = k
对绕同一中心天体运动的行星/卫星成立

k 为常量

注意：

同一中心天体下才能直接比较

中学阶段常按圆轨道处理

6. 天体质量与密度的计算
6.1 由环绕速度求中心天体质量
text
G*Mm / r^2 = mv^2 / r
得：

text
M = v^2*r / G
6.2 由周期求中心天体质量
text
G*Mm / r^2 = m*4π^2*r / T^2
得：

text
M = 4π^2*r^3 / (G*T^2)
6.3 第一宇宙速度
text
v1 = sqrt(GM / R)
v1 = sqrt(gR)
地球表面约为：

text
v1 ≈ 7.9 km/s
6.4 由第一宇宙速度求密度
若天体近似为球体：

text
M = 4/3*π*R^3*ρ
可得：

text
ρ = 3v1^2 / (4πGR^2)
7. 宇宙速度
7.1 第一宇宙速度
物体贴近天体表面做圆周运动所需的最小速度

text
v1 = sqrt(GM / R) = sqrt(gR)
7.2 第二宇宙速度
脱离地球引力束缚所需的最小速度

text
v2 = sqrt(2) * v1
地球约为：

text
v2 ≈ 11.2 km/s
7.3 第三宇宙速度
脱离太阳系所需的最小速度

text
v3 ≈ 16.7 km/s
注意：

这些是“最低速度”，不是“飞行速度上限”

8. 人造卫星
卫星做匀速圆周运动时：

text
G*Mm / r^2 = mv^2 / r
因此：

text
v = sqrt(GM / r)
T = 2π*sqrt(r^3 / (GM))
a = GM / r^2
结论：

轨道半径越大，速度越小

周期越大

向心加速度越小

万有引力越小

三、重要知识点总结
1. 运动学三大核心
位移、速度、加速度

匀变速直线运动公式

自由落体和竖直上抛可看作匀变速运动

2. 牛顿运动定律核心
合力决定加速度

力不是维持运动的原因

受力分析是所有题目的基础

3. 曲线运动核心思想
速度方向沿切线

运动分解为两个互不影响的直线运动

向心力只负责改变速度方向，不一定改变速度大小

4. 万有引力核心思想
天体之间普遍存在相互吸引

万有引力既是引力，也是向心力来源

轨道问题常用“万有引力 = 向心力”

四、考试高频易错点
1. 公式使用前先判断运动类型
匀变速直线运动才能直接用 v = v0 + at

平抛运动必须分解

圆周运动要看是否有向心力

2. 受力分析不要混淆“力”和“合力”
牛顿第二定律里的是合外力

作用力与反作用力不相互抵消

3. 注意单位统一
常见单位：

长度：m

质量：kg

时间：s

力：N

速度：m/s

加速度：m/s^2

4. 圆周运动里最容易错的点
向心力不是单独一种力

向心加速度方向始终指向圆心

线速度和角速度不要混淆

5. 万有引力题目的关键
r 是质心间距离

天体表面附近常取 r = R

轨道半径变大，卫星速度变小，周期变大

6. 自由落体与竖直上抛
只要不计空气阻力，都属于加速度恒为 g 的匀变速运动

上抛上升和下降过程的加速度方向始终向下
