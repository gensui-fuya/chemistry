# 📐 高一数学·必修二第一章《三角函数》月考复习笔记（北师大版）


## 📖 一、基础知识概览

三角函数是高中数学的重点章节，主要研究任意角的三角函数定义、性质、图像及其应用。第一次月考范围通常涵盖**任意角与弧度制、三角函数的定义、诱导公式、三角函数的图像与性质（正弦、余弦、正切）** ，以及简单的三角恒等变换[reference:0]。


## 📐 二、角的概念与推广

### 1. 角的基本概念

角可以看成平面内一条射线绕着端点从一个位置旋转到另一个位置所成的图形。

**角的分类：**

- **正角**：一条射线绕其端点按逆时针方向旋转形成的角
- **负角**：一条射线绕其端点按顺时针方向旋转形成的角
- **零角**：一条射线没有作任何旋转形成的角

> 📌 **注意**：角的正负由旋转方向决定，逆时针为正，顺时针为负。

### 2. 平面直角坐标系中的角

在直角坐标系中，角的顶点与原点重合，角的始边与 x 轴的非负半轴重合。

- **象限角**：角的终边落在第几象限，就称这个角为第几象限角
- **轴线角**：角的终边落在坐标轴上的角

> 📌 **注意**：象限角的判断关键是看**终边所在位置**，而非角的大小本身。

### 3. 终边相同的角

所有与角 α 终边相同的角，连同角 α 在内，可构成一个集合：

**S = { β | β = α + k·360°, k ∈ Z }**

在弧度制下：**S = { β | β = α + 2kπ, k ∈ Z }**

> 📌 **注意**：
> - 终边相同的角不一定相等（如 0° 与 360° 终边相同，但不相等）
> - 两个角终边相同 ⇔ 它们的差是 360° 的整数倍


## 📏 三、弧度制

### 1. 弧度制的定义

在单位圆中，把长度等于 1 的弧所对的圆心角称为 **1 弧度的角**，记作 1 rad（通常省略不写）。这种以弧度作为单位来度量角的方法，称作**弧度制**[reference:1]。

**核心理解**：在单位圆中，弧长就等于圆心角的弧度数。半径为 r 的圆中，弧度数 |α| = 弧长 / 半径。

### 2. 角度与弧度的换算

| 换算方向 | 公式 |
|---------|------|
| 角度 → 弧度 | 1° = π/180 rad |
| 弧度 → 角度 | 1 rad = (180/π)° |

**基本换算关系**：**180° = π rad**

### 3. 特殊角的度数与弧度对应表

| 角度 | 0° | 30° | 45° | 60° | 90° | 120° | 135° | 150° | 180° | 270° | 360° |
|------|----|----|----|----|----|-----|------|------|------|------|------|
| 弧度 | 0 | π/6 | π/4 | π/3 | π/2 | 2π/3 | 3π/4 | 5π/6 | π | 3π/2 | 2π |

### 4. 弧长公式与扇形面积公式

设扇形半径为 R，弧长为 l，圆心角的弧度数为 α（α > 0），则：

- **弧长公式**：**l = α·R**
- **扇形面积公式**：**S = ½·l·R = ½·α·R²**

> 📌 **注意**：上述公式中的圆心角 α 必须采用**弧度制**，不能使用角度制。

### 5. 常见误区提示

- 角度制与弧度制是两种不同的度量方式，不能混用（如“360° = 2π”这种写法本身就是错误的，应是“360° = 2π rad”）
- 正角的弧度数是正数，负角的弧度数是负数，零角的弧度数是 0[reference:2]


## 🔺 四、任意角的三角函数定义

### 1. 单位圆定义法

设 α 是一个任意角，它的终边与单位圆（半径为 1 的圆）交于点 **P(x, y)**，则：

| 函数 | 定义 | 记法 |
|------|------|------|
| 正弦函数 | 点 P 的纵坐标 y | sin α = y |
| 余弦函数 | 点 P 的横坐标 x | cos α = x |
| 正切函数 | 点 P 的纵坐标与横坐标之比 | tan α = y/x (x ≠ 0) |

正弦、余弦、正切都是以角为自变量，以单位圆上的点的坐标或坐标的比值为函数值的函数，统称为**三角函数**[reference:3]。

### 2. 终边上任意点定义法（推广）

设 P(x, y) 是角 α 终边上异于原点的任一点，它到原点 O 的距离 r = √(x² + y²)，则：

| 函数 | 表达式 |
|------|--------|
| sin α = y/r |
| cos α = x/r |
| tan α = y/x (x ≠ 0) |

> 📌 **注意**：三角函数值只与角 α 终边的位置有关，与终边上点 P 的选取无关[reference:4]。


## 🎯 五、三角函数值在各象限的符号

记忆口诀：**“一全正，二正弦，三正切，四余弦”**

| 象限 | sin α | cos α | tan α |
|------|-------|-------|-------|
| 第一象限 | + | + | + |
| 第二象限 | + | - | - |
| 第三象限 | - | - | + |
| 第四象限 | - | + | - |

> 📌 **理解**：即——第一象限全为正；第二象限正弦为正；第三象限正切为正；第四象限余弦为正[reference:5]。

| 角度 | 弧度 | sin | cos | tan |
|------|------|-----|-----|-----|
| 0° | 0 | 0 | 1 | 0 |
| 30° | π/6 | 1/2 | √3/2 | √3/3 |
| 45° | π/4 | √2/2 | √2/2 | 1 |
| 60° | π/3 | √3/2 | 1/2 | √3 |

## 🔁 七、诱导公式

### 1. 诱导公式列表

| 编号 | 公式 |
|------|------|
| ① | sin(α + 2kπ) = sin α，cos(α + 2kπ) = cos α，tan(α + 2kπ) = tan α (k ∈ Z) |
| ② | sin(-α) = -sin α，cos(-α) = cos α，tan(-α) = -tan α |
| ③ | sin(π - α) = sin α，cos(π - α) = -cos α，tan(π - α) = -tan α |
| ④ | sin(π + α) = -sin α，cos(π + α) = -cos α，tan(π + α) = tan α |
| ⑤ | sin(π/2 - α) = cos α，cos(π/2 - α) = sin α |
| ⑥ | sin(π/2 + α) = cos α，cos(π/2 + α) = -sin α |
| ⑦ | sin(3π/2 - α) = -cos α，cos(3π/2 - α) = -sin α |
| ⑧ | sin(3π/2 + α) = -cos α，cos(3π/2 + α) = sin α |

### 2. 记忆口诀——“奇变偶不变，符号看象限”

- **奇变偶不变**：对于“k·π/2 ± α”形式的诱导公式：
  - 若 k 为偶数，则函数名不变（sin 仍为 sin，cos 仍为 cos）
  - 若 k 为奇数，则函数名改变（sin ↔ cos，tan ↔ cot）
- **符号看象限**：将 α 视为**锐角**，判断原函数在相应象限的符号，决定结果的正负

### 3. 应用步骤——“负化正，大化小，小化锐，锐求值”

1. **负角化正角**：利用 sin(-α) = -sin α，cos(-α) = cos α
2. **大角化小角**：减去 2π 的整数倍
3. **小角化锐角**：利用诱导公式转化到 0°~90° 范围
4. **查表求值**[reference:6]

> 📌 **注意**：在应用“符号看象限”时，必须始终把 α 看作**锐角**来判定，而非 α 实际的值[reference:7]。


## 📈 八、三角函数的图像与性质

### 1. 正弦函数 y = sin x 的图像与性质

**图像特征**：波浪形曲线，关于原点对称（奇函数）

| 性质 | 内容 |
|------|------|
| 定义域 | R（全体实数） |
| 值域 | [-1, 1] |
| 周期性 | 最小正周期 T = 2π |
| 奇偶性 | 奇函数（sin(-x) = -sin x） |
| 递增区间 | [-π/2 + 2kπ, π/2 + 2kπ] (k ∈ Z) |
| 递减区间 | [π/2 + 2kπ, 3π/2 + 2kπ] (k ∈ Z) |
| 最大值 | 1（当 x = π/2 + 2kπ 时） |
| 最小值 | -1（当 x = 3π/2 + 2kπ 时） |
| 对称轴 | x = π/2 + kπ |
| 对称中心 | (kπ, 0) |

### 2. 余弦函数 y = cos x 的图像与性质

**图像特征**：波浪形曲线，关于 y 轴对称（偶函数）（可由 y = sin x 向左平移 π/2 个单位得到）

| 性质 | 内容 |
|------|------|
| 定义域 | R |
| 值域 | [-1, 1] |
| 周期性 | 最小正周期 T = 2π |
| 奇偶性 | 偶函数（cos(-x) = cos x） |
| 递增区间 | [-π + 2kπ, 2kπ] (k ∈ Z) |
| 递减区间 | [2kπ, π + 2kπ] (k ∈ Z) |
| 最大值 | 1（当 x = 2kπ 时） |
| 最小值 | -1（当 x = π + 2kπ 时） |
| 对称轴 | x = kπ |
| 对称中心 | (π/2 + kπ, 0) |

### 3. 正切函数 y = tan x 的图像与性质

**图像特征**：间断的曲线，被直线 x = π/2 + kπ 分隔（渐近线），关于原点对称（奇函数）

| 性质 | 内容 |
|------|------|
| 定义域 | { x \| x ≠ π/2 + kπ, k ∈ Z } |
| 值域 | R（全体实数） |
| 周期性 | 最小正周期 T = π |
| 奇偶性 | 奇函数（tan(-x) = -tan x） |
| 单调性 | 在每个开区间 (-π/2 + kπ, π/2 + kπ) 内单调递增 |
| 对称中心 | (kπ/2, 0) |
| 渐近线 | x = π/2 + kπ |

> 📌 **特别注意**：
> - 正切函数的定义域不能取 π/2 + kπ（即 90° + 180°·k）[reference:8]
> - 正切函数**没有对称轴**，只有对称中心[reference:9]
> - 正切函数在整个定义域内**不具有单调性**，只能说在每个单调区间内单调递增

### 4. “五点法”作图

作 y = sin x，x ∈ [0, 2π] 的图像时，五个关键点：

| 点 | x | sin x |
|----|---|-------|
| ① | 0 | 0 |
| ② | π/2 | 1 |
| ③ | π | 0 |
| ④ | 3π/2 | -1 |
| ⑤ | 2π | 0 |

余弦函数 y = cos x 的五点取法类似，对应点为 (0,1), (π/2,0), (π,-1), (3π/2,0), (2π,1)[reference:10]。


## 🔄 九、函数 y = A sin(ωx + φ) 的图像与性质

这是三角函数的“综合形态”，是月考的**压轴考点**，必须重点掌握。

### 1. 参数 A, ω, φ 的物理意义

| 参数 | 名称 | 含义 |
|------|------|------|
| A | 振幅 | 表示振动时离开平衡位置的最大距离，决定值域和最大值、最小值 |
| ω | 角频率 | 决定周期（周期 T = 2π/|ω|），ω > 0 时频率 f = ω/(2π) |
| φ | 初相 | 当 x = 0 时的相位，决定图像的左右平移 |
| ωx + φ | 相位 | 反映振动的“进程” |

**重要关系**：
- 值域：[-A, A]（假设函数为 y = A sin(ωx + φ) 且无垂直平移）
- 周期：**T = 2π/|ω|**
- 频率：f = 1/T = |ω|/(2π)[reference:11]

> 📌 **注意**：振幅 A 一定为正数；当函数形式为 y = A sin(ωx + φ) + B 时，值域为 [-A + B, A + B]。

### 2. 图像变换（由 y = sin x 到 y = A sin(ωx + φ)）

变换路径（按“先平移后伸缩”顺序）：

1. **相位变换**：y = sin x → y = sin(x + φ)（向左平移 |φ| 个单位，φ > 0 向左，φ < 0 向右）
2. **周期变换**：y = sin(x + φ) → y = sin(ωx + φ)（横坐标伸缩为原来的 1/ω 倍）
3. **振幅变换**：y = sin(ωx + φ) → y = A sin(ωx + φ)（纵坐标伸缩为原来的 A 倍）[reference:12]

> 📌 **易错点**：
> - 变换顺序不同时，平移量不同！先周期后相位时，平移量应改为 |φ/ω|[reference:13]
> - 图像变换是对 **x** 进行的变换，不是对“角”进行的变换
> - 记忆谐音：“**想欺负**”（相 → 相位变换，期 → 周期变换，幅 → 振幅变换）

### 3. 重要结论

- 求周期时，先将函数化为 y = A sin(ωx + φ) 形式，再用公式 T = 2π/|ω|
- 求对称轴：令 ωx + φ = π/2 + kπ，解出 x
- 求对称中心：令 ωx + φ = kπ，解出 x，对称中心为 (x, 0)
- 研究单调性时，设 u = ωx + φ，先求 u 的范围，再结合 sin u 的单调性[reference:14]


## ✨ 十、同角三角函数的基本关系

### 1. 平方关系

**sin²α + cos²α = 1**

> 由单位圆定义直接可得：x² + y² = 1 ⇒ cos²α + sin²α = 1

### 2. 商数关系

**tan α = sin α / cos α**（cos α ≠ 0）

### 3. 倒数关系

**tan α·cot α = 1**（tan α 和 cot α 均存在时）

### 应用示例

已知 sin α，求 cos α 时：

cos α = ±√(1 - sin²α)（符号由 α 所在象限确定）

> 📌 **注意**：开方后符号的确定取决于 α 的象限！不可遗漏符号。


## ⚠️ 十一、月考高频易错点与答题技巧

### 1. 角的概念与弧度制
- ❌ **误区**：认为“第一象限角一定是锐角”
- ✅ **正解**：第一象限角是 { α | 0° + k·360° < α < 90° + k·360°, k ∈ Z }，k 可以取其他整数（如 400° 也在第一象限），因此第一象限角不一定小于 90°[reference:15]
- ❌ **误区**：角度与弧度混用（如“π/2 = 90°”混写在一个式子中）
- ✅ **正解**：角度制与弧度制不能混用，要么全部使用角度制，要么全部使用弧度制

### 2. 三角函数符号
- ❌ **误区**：死记符号公式，不结合象限分析
- ✅ **正解**：利用“一全正，二正弦，三正切，四余弦”的口诀，先判断角所在象限，再确定符号

### 3. 诱导公式
- ❌ **误区**：错误理解“奇变偶不变”——误以为奇偶指 α 的系数是奇数还是偶数
- ✅ **正解**：“奇”和“偶”指的是 π/2 的倍数 k，不是 α 的系数
- ❌ **误区**：应用“符号看象限”时把 α 当作实际值
- ✅ **正解**：始终把 α **看作锐角**来判定符号，不要被 α 的实际大小迷惑[reference:16]

### 4. 函数性质
- ❌ **误区**：认为正切函数在整个定义域内单调递增
- ✅ **正解**：正切函数只在每个单调区间 (-π/2 + kπ, π/2 + kπ) 内单调递增，但由于定义域不连续，不能说在整个定义域上单调[reference:17]
- ❌ **误区**：求周期时忘记加绝对值（ω 可能为负数）
- ✅ **正解**：周期 T = 2π/|ω|，不是 2π/ω

### 5. 图像变换
- ❌ **误区**：先伸缩后平移时平移量计算错误
- ✅ **正解**：先伸缩后平移时，平移量应为 φ/ω（而不是 φ）[reference:18]

### 6. 定义域问题
- ❌ **误区**：只考虑函数式有意义，忽略三角函数自身定义域要求
- ✅ **正解**：求复合三角函数定义域时，要同时考虑内层函数和外层函数的定义域限制[reference:19]


## 📝 十二、常用公式速查

### 1. 基本公式
- sin²α + cos²α = 1
- tan α = sin α / cos α
- cot α = cos α / sin α = 1 / tan α

### 2. 诱导公式核心记忆法
- 函数名：k·π/2 ± α（k 为偶数→不变，k 为奇数→变）
- 符号：α 看作锐角，原函数值在相应象限的符号

### 3. 周期公式
- y = sin(ωx + φ)：T = 2π/|ω|
- y = cos(ωx + φ)：T = 2π/|ω|
- y = tan(ωx + φ)：T = π/|ω|

|------|---------|
| 求三角函数值 | 先判断角所在象限（定符号）→ 利用公式计算 |
| 化简求值 | “负化正→大化小→小化锐→锐求值”四步走 |
| 比较大小 | 统一到同一单调区间内进行比较，或利用单位圆中的三角函数线 |

| 求解析式 | 利用五点法或图像变换规律，结合已知条件建立方程 |
| 函数性质分析 | 化为 y = A sin(ωx + φ) 形式后套用性质公式 |
